LGP - Linear Genetic Programming
===

Una implementación para encontrar funciones de estimación utilizando Programación Genética Lineal.

* LgpPython, implementación básica hecha en Python.
* LgpCpp, implementación utilizando C++ y OpenMp para paralelización.
* MathModel, implementación de un modelo matemático de estimación, contra el que se realizan comparaciones de las funciones de estimación encontradas.
